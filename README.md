# Taller de Nivelación

Este repositorio contiene:

- Archivo teórico de Git (git.md)
- Archivo teórico de Programación (programacion.md)
- 3 ejercicios prácticos en Java:

  - Contador de vocales y consonantes

## Cómo ejecutar

1. Compilar:
javac NombreArchivo.java

2. Ejecutar:
java NombreArchivo
